    Download Python 3.6.5 64-bit installer
    https://www.python.org/ftp/python/3.6.5/python-3.6.5-amd64.exe
    Als Administrator ausführen
    Install für ungedüldige
        „Install Now“ + „Add Python 3.6 to Path“ Anhaken
    Custom Install „Customize Installation“ auswählen
        Optional Features  Alles angehakt lassen
        Advanced Options  Add Python to environment variables anhaken
        Customize install location  z.B. C:\Program Files\Python3
    Install!
    CMD aufmachen, „py“ angeben, python 3.6.5 erscheint...
